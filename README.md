# 폴더 감시 프로그램

Windows용 폴더 내 파일 변경/수정/삭제를 실시간으로 감시하는 데스크톱 애플리케이션입니다.

## 기능

- 여러 폴더 동시 감시
- 파일 생성/수정/삭제 실시간 감지
- 변경 로그 기록 (최근 100건)
- 시스템 트레이 지원 (백그라운드 실행)
- 설정 자동 저장

## 설치 방법

### 방법 1: 설치 프로그램 사용

1. `dist` 폴더의 `폴더 감시 프로그램 Setup 1.0.0.exe` 실행
2. 설치 경로 선택 후 설치
3. 바탕화면 또는 시작 메뉴의 "폴더 감시 프로그램" 아이콘 클릭

### 방법 2: 포터블 버전

1. `dist` 폴더의 `폴더 감시 프로그램 1.0.0.exe` 실행
2. 설치 없이 바로 사용 가능

### 방법 3: 개발 환경에서 실행

```bash
# 패키지 설치
npm install

# 실행
npm start
```

## 빌드 방법

```bash
# 패키지 설치
npm install

# 설치 파일 생성
npm run build

# 포터블 버전만 생성
npm run build:portable
```

빌드 후 `dist` 폴더에 파일 생성:
- `폴더 감시 프로그램 Setup 1.0.0.exe` - 설치 프로그램
- `폴더 감시 프로그램 1.0.0.exe` - 포터블 버전

## 사용 방법

### 폴더 추가

1. 감시할 폴더 경로 입력 (예: `C:\Users\Documents`)
2. "추가" 버튼 클릭

### 폴더 삭제

1. 감시 중인 폴더 목록에서 "삭제" 버튼 클릭

### 로그 확인

- 변경 로그 섹션에서 실시간으로 파일 변경 내역 확인
- 로그는 최근 100건까지 유지
- "로그 지우기" 버튼으로 로그 초기화

### 트레이 사용

- 창 닫기(X) 버튼 클릭 시 트레이로 최소화
- 트레이 아이콘 더블클릭으로 창 열기
- 트레이 아이콘 우클릭 → "종료"로 프로그램 완전 종료

## 기술 스택

- Electron 28
- Node.js (fs.watch API)
- HTML/CSS/JavaScript

## 주의사항

- 네트워크 드라이브 감시는 지원되지 않을 수 있습니다
- 대용량 폴더 감시 시 시스템 리소스 사용량이 증가할 수 있습니다
- 포트 4400을 사용합니다 (충돌 시 자동으로 기존 프로세스 종료)
